\# MINI STREETWEAR



一個以韓系極簡風格為主的服飾電商前端專案。



\## 🔹 專案功能

\- 商品列表（JavaScript 動態渲染）

\- 商品圖片顯示

\- 尺寸與數量選擇

\- 購物車功能（LocalStorage）

\- 即時計算總金額

\- Lookbook 視覺頁

\- 手機版 RWD 支援



\## 🔹 使用技術

\- HTML5

\- CSS3（Flex / Grid / RWD）

\- JavaScript（DOM 操作、LocalStorage）



\## 🔹 專案特色

\- 無框架純前端實作

\- 結構清楚、易於擴充

\- 適合作為品牌官網或電商前端範例
